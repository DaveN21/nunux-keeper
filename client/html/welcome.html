NUNUX :: Keeper

keep the best of your internet surfing in one place.

  • Archive

    View, edit and archive for ever many kind of internet documents in your own storage.

  • Classify and search

    Classify your documents and retrieve them thanks to a powerful search engine.

  • Free & open source

    This software free and under the GPLv3 licence.

Features

  • Categorize your documents

    Create custom category and classify your documents.

  • Create new document

    Import new document from anywhere. Create your own, or extract it from an URL.

  • Full text search

    Retrieve a specific document thanks to a powerful query system.

  • Use the powerful Bookmarklet

    Save whole the web page or just a part while surfing.

  • WYSIWYG

    Edit the document as you view it.

  • Keyboard shortcuts

    Use keyboard shortcuts to be more efficient. Type '?' and see the magic appen.

  • Clean API

    Create your own client or service thanks to a complete RESTFul JSON API.

  • 100% Javascript

    One language to rule them all!
